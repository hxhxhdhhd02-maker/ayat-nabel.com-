# Mobile App

This is the React Native mobile application for the Admin Dashboard.

## Setup

1.  Navigate to the mobile directory:
    ```bash
    cd mobile
    ```

2.  Install dependencies (if not already done):
    ```bash
    yarn install
    ```

## Running the App

To start the Expo development server:

```bash
yarn start
```

Scan the QR code with the **Expo Go** app on your Android or iOS device.

## Features

- **Admin Login**: Secure login for the teacher account.
- **Dashboard**: Overview of student statistics and activities.
