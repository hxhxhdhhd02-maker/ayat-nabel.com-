# دليل إعداد منصة الأستاذة آيات نبيل التعليمية

## نظرة عامة
منصة تعليمية متكاملة لتدريس اللغة الإنجليزية للمراحل الإعدادية والثانوية مع نظام إدارة محتوى كامل للمدرسة.

## المميزات الرئيسية

### للطلاب:
- تسجيل الدخول برقم الموبايل والصف الدراسي
- عرض الكورسات المتاحة للصف الدراسي
- مشاهدة المحاضرات المفعلة
- تتبع التقدم في كل محاضرة
- تحميل ملفات PDF المرفقة مع المحاضرات
- مشغل فيديو متقدم مع التحكم في السرعة والجودة

### للمدرسة:
- لوحة تحكم شاملة لإدارة المحتوى
- إنشاء وإدارة الكورسات حسب الصفوف الدراسية
- إضافة محاضرات من اليوتيوب
- رفع ملفات PDF للمحاضرات
- البحث عن الطلاب وتفعيل الكورسات لهم
- تحديد سعر لكل كورس

## إعداد حساب المدرسة

لإنشاء حساب مدرسة، تحتاج إلى إضافة المدرسة يدوياً إلى قاعدة البيانات:

### الخطوات:

1. سجل دخول إلى Supabase Dashboard
2. انتقل إلى SQL Editor
3. قم بتنفيذ الأمر التالي (استبدل البيانات بالبيانات الفعلية):

```sql
-- أولاً: إنشاء مستخدم في نظام Authentication
-- يجب القيام بذلك من خلال Supabase Dashboard > Authentication > Users > Add User
-- Email: [رقم_الموبايل]@platform.local
-- Password: [كلمة_المرور_التي_تريدها]
-- مثال: 01234567890@platform.local

-- ثانياً: بعد إنشاء المستخدم، احصل على الـ UUID الخاص به من صفحة Users

-- ثالثاً: أضف البيانات إلى جدول profiles
INSERT INTO profiles (id, phone_number, full_name_arabic, grade, role)
VALUES (
  '[UUID من خطوة 2]',
  '01234567890',
  'آيات نبيل',
  'اول_اعدادي',  -- يمكن اختيار أي صف
  'teacher'
);
```

### مثال كامل:
إذا كان UUID المستخدم: `123e4567-e89b-12d3-a456-426614174000`

```sql
INSERT INTO profiles (id, phone_number, full_name_arabic, grade, role)
VALUES (
  '123e4567-e89b-12d3-a456-426614174000',
  '01012345678',
  'آيات نبيل',
  'اول_اعدادي',
  'teacher'
);
```

## استخدام المنصة

### تسجيل دخول الطلاب:
1. افتح المنصة
2. اختر "إنشاء حساب"
3. أدخل:
   - الاسم بالعربي
   - رقم الموبايل
   - الصف الدراسي
   - كلمة المرور
4. انقر "إنشاء حساب"

### تسجيل دخول المدرسة:
1. افتح المنصة
2. اختر "تسجيل الدخول"
3. أدخل رقم الموبايل وكلمة المرور
4. سيتم توجيهك تلقائياً إلى لوحة التحكم

## سير العمل

### إضافة كورس جديد:
1. من لوحة تحكم المدرسة، انقر "كورس جديد"
2. أدخل:
   - عنوان الكورس
   - الوصف
   - الصف الدراسي المستهدف
   - السعر
   - رابط صورة الغلاف (اختياري)
3. انقر "إنشاء الكورس"

### إضافة محاضرات:
1. من قائمة الكورسات، انقر "المحاضرات" على الكورس المطلوب
2. انقر "محاضرة جديدة"
3. أدخل:
   - عنوان المحاضرة
   - الوصف
   - رابط فيديو اليوتيوب
   - رابط ملف PDF (اختياري)
   - مدة الفيديو بالثواني
4. انقر "إضافة المحاضرة"

### تفعيل كورس لطالب:
1. انقر على تبويب "الطلاب"
2. ابحث عن الطالب باسمه أو رقم موبايله
3. انقر "تفعيل كورس"
4. اختر الكورس المطلوب من القائمة
5. انقر "تفعيل"

## ملاحظات هامة

- يجب أن يكون رابط اليوتيوب بصيغة صحيحة (مثال: https://www.youtube.com/watch?v=VIDEO_ID)
- ملفات PDF يجب أن تكون مرفوعة على خدمة استضافة ملفات (مثل Google Drive، Dropbox، أو خدمة تخزين سحابي)
- الطالب لن يتمكن من مشاهدة محاضرات الكورس إلا بعد تفعيله من قبل المدرسة
- يتم حفظ تقدم الطالب تلقائياً كل 10 ثوان أثناء مشاهدة الفيديو
- يمكن للطالب التحكم في سرعة الفيديو من خلال إعدادات المشغل
